# ts-client-demo


### How to run on browser

Install http server

```bash
 $ npm install http-server -g
```

Install declaration files dependencies

```bash
 $ npm install
```

Compile ts

```bash
 $ tsc
```

Start http server

```bash
 $ http-server -p 8081
```

Go to `localhost:8081` from your browser. (Make sure your ts-sample-api is runnint on `8080`)
